# HSpiceLab
MATLAB toolbox to read HSPICE generated output files

## read_hspice_tr.m  
Read data from HSPICE generated .tr# files (ASCII formatted).  
For a note, ASCII formatted file can be created using .OPTION post=2 in the .sp file.
